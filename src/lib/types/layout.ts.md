# layout.ts

## 핵심 개념 (Core Concept)
애플리케이션의 레이아웃과 관련된 모든 타입 정의를 포함하는 파일입니다. 섹션(Section), 모달(Modal), 레이아웃 상태(LayoutState), 뷰 상태(ViewState) 등 UI 구성 요소들의 데이터 구조를 정의합니다.
