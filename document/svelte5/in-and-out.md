# in: and out:

### On this page

-   [in: and out:](in-and-out.html)

The `in:` and `out:` directives are identical to [`transition:`](transition.html), except that the resulting transitions are not bidirectional — an `in` transition will continue to ‘play’ alongside the `out` transition, rather than reversing, if the block is outroed while the transition is in progress. If an out transition is aborted, transitions will restart from scratch.

<script>
  import { fade, fly } from 'svelte/transition';
  let visible = $state(false);
</script>
<label>
  <input type="checkbox" bind:checked={visible}>
  visible
</label>
{#if visible}
	<div in:fly={{ y: 200 }} out:fade>flies in, fades out</div>
{/if}
