# <svelte:body>

### On this page

-   [<svelte:body>](svelte-body.html)

<svelte:body onevent={handler} />

Similarly to `<svelte:window>`, this element allows you to add listeners to events on `document.body`, such as `mouseenter` and `mouseleave`, which donâ€™t fire on `window`. It also lets you use [actions](use.html) on the `<body>` element.

As with `<svelte:window>` and `<svelte:document>`, this element may only appear the top level of your component and must never be inside a block or element.

<svelte:body onmouseenter={handleMouseenter} onmouseleave={handleMouseleave} use:someAction />
